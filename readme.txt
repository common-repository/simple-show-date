== Installation ==

This section describes how to install the plugin and get it working.

1. Upload simple-show-date folder to your wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to setting page and choose your shortcode.
4. Copy shortcode and paste where you want to show date, That’s All

== Frequently Asked Questions ==

= How can i add today date on my website? =

You simply have to install plugin then activate and take shortcode from setting page, that’s all

= How can i show yesterday date? =

You can go to setting page and see you will get -day or + day option.

= Can i add custom date format? =

Yes, just go to setting page and choose custom style, and you can see code for day, month, year etc.

= Can my date visible on desktop & mobile both? =

Yes, by default it will visible on mobile & desktop both, if you want to write some css, you can write with class name.
